# Pizza Shop API

This api let's you create users to make pizza orders securely

### What's done :

Entities :
* Pizza -- name, price
* Order -- owner, pizzas, createdAt, updatedAt, state, isSold
* User -- username, email, password, imageUrl, orders

Auth :
* Jwt secured Authentication
